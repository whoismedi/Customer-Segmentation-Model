A customer segmentation project using K-Means unsupervised machine learning algorithm to group customers based on similarity, with the Elbow Method applied to determine the optimal number of clusters and perform effective customer clustering.

**Technologies used:**

- scikit-learn
- pandas
- numpy
- matplotlib
- seaborn
- streamlit (for UI)
